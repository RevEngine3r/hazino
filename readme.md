# Hazino - ردیاب درآمد و هزینه 📊

[**English Version**](README-en.md) | نسخه فارسی

Hazino یه اپ مدرن و خوش‌دسته که می‌تونه درآمدها و هزینه‌های شما رو ساده و مرتب دنبال کنه.

## ✨ چه کارایی می‌کنه

* **ثبت ساده**: درآمدها و هزینه‌ها رو راحت ثبت کنید.
* **خلاصه لحظه‌ای**: در یه نگاه، کل درآمدها، هزینه‌ها و موجودی خودتون رو ببینید.
* **رنگ‌بندی هوشمند**: با رنگ‌های مختلف، وضعیت مالی خودتون رو چک کنید.
* **طراحی مدرن**: ظاهری تم و مینیمال مثل اپ‌های گوگل.
* **حالت روز و شب**: با تنظیم تم گوشی شما هماهنگ می‌شه.

## 🚀 شروع کنیم

1. مخزن رو کلون کنید: `git clone https://github.com/your-username/Hazino.git`
2. پروژه رو با ابزار موردنظرتون باز کنید.
3. بسازید و اجرا کنید.

### دانلود APK

می‌تونید آخرین نسخه APK رو از بخش [Releases](https://github.com/your-username/Hazino/releases)
دانلود کنید.

## 🛠️ چه چیزایی استفاده کرده

* **زبان**: کاتلین
* **چارچوب**: Jetpack Compose

## 📋 چطور کار می‌کنه

* روی دکمه + (پایین صفحه) بزنید تا یه تراکنش جدید وارد کنید.
* از صفحه باز شونده، نوع تراکنش (درآمد یا هزینه) رو انتخاب کنید.
* جزئیات رو وارد کنید و ذخیره کنید.
* خلاصه وضعیت مالیتون رو در صفحه اصلی ببینید.

## 🎨 چه ویژگی‌های طراحی‌ای داره

* **طراحی مینیمال**: مثل اپ‌های گوگل، تمیز و ساده.
* **راحت و شهودی**: دکمه‌ها و امکاناتش واضح و آشکاره.
* **هم‌خوانی با حالت گوشی**: با تم روز یا شب گوشی شما کار می‌کنه.
* **دو زبانه**: هم فارسی (راست‌چین) و هم انگلیسی (چپ‌چین) پشتیبانی می‌شه.
